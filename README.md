# ToDo-Manager
 
> Eine einfach Single Page Application (SAP) für die Verwaltung von ToDos
 
**[Page öffnen](https://manuellindinger.ch/ToDo-Manager/)**
 
---
 
## Details
 
ToDo-Manager ist eine einfache, clientseitige ToDo Liste, welche vollständig im Browser läuft. Daten werden nur lokal im Browser gespeichert. - Kein Login nötig
 
- Verwaltung von ToDos (CRUD)
- Speicherung im localStorage
---

| Was | Wie |
|---|---|
| Aufgabe hinzufügen | Menü über den Button öffnen, Details eingeben und auf „Hinzufügen“ klicken |
| Aufgabe erledigen | Fortschritt aktualisieren |
| Aufgabe löschen | Auf den Lösch-Button klicken |
| Suchen | Suchbegriff in die Suchleiste eingeben, optional Filter oder Sortierung verwenden |
